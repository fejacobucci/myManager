Definições de Projeto

	> Título: Nome o projeto.
	> Previsão de Início: A data prevista para início das atividades do projeto.
	> Previsão de Término: Data prevista para que o projeto seja encerrado com todas as etapas concluídas.
	> Tarefas do Projeto: Quais serão as atividades para que o projeto seja executado.
		>> Data de início da tarefa.
		>> Data de término da tarefa.
		>> Antecessores para execução.
		>> Prioridade da tarefa.
		>> Tempo para execução da tarefa.
		>> Executor da tarea.
		>> Percentual representativo da tarefa.
		>> Status a tarefa.
			[
				P - Pendente, 
				E - Em Execução, 
				C - Concluída, 
				A - Aguardando Informaçoes
			]	
	> Envolvidos: Todos as pessoas do próprio time envolvidas no projeto.
	> Comentários do projeto.
	
	
Ciclo(Circuito) do Projeto

	> Tarefa: Qual tarefa a ser executada.
	> Executor: Quem irá executar a tarefa.
	> Início: Data em que a tarefa erá início.
	> Fim: Data da entrega da tarefa atribuída
	> Status da tarefa
		[
			A - Atribuida.
			E - Executando.
			C - Concluida.
			P - Pendente.
			! - Cancelada.
		]
	> Antecessor: Qual tarefa ou atividade deve ser executada antes.
	> Sucessor: Qual a próxima tarefa após a conclusão.
	> Comentários
